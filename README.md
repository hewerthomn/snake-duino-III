#Snake Duino III
--------------

Arduino Snake game controlled by N64 Controller

Video https://vimeo.com/80725135


-----------
>##Components
> - Nokia 5110 LCD attached to pins 7, 6, 5, 4, 3
> - Active Buzzer attached to pin 8
> - N64 Controller attached to pin 2
